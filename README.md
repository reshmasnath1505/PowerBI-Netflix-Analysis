OVERVIEW -
This project performs a descriptive analysis of Netflix’s content library to understand how movies and TV shows are distributed across genres, ratings, release years, and countries. The goal is to uncover meaningful patterns and trends that highlight Netflix’s global content strategy and growth over the years.

DATASET-

Source: Netflix Titles Dataset (commonly available via Kaggle or publicly shared repositories)

Contains information on:

Title type (Movie/TV Show)

Release year

Country of production

Rating (U/A, PG, TV-MA, etc.)

Genre listed as “listed_in”

Duration (minutes or seasons)

Description


TOOLS AND TECHNOLOGIES - 

Power BI 

Data Cleaning: Power Query

Visualization: Power BI
